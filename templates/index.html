PyPml2JSON Homepage

Parses OPML as HTML to then convert to JSON using the bs2json library.

  • Input: OPML String
  • Output: JSON file (if available)

Site Map

  • /API/template/ - Query directly by image name in the Fir-Functions Google Cloud Bucket Database
  • /API/template?name= - Query by image name via GET request in the Fir-Functions Google Cloud Bucket Database
  • /API/template?link= - Submit your own image address URL
